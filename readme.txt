Module ini adalah addon dari billingotomatis gateway sebagai SMS gateway WooCommerce wordpress.

== Description ==

Billingotomatis.com merupakan penyedia layanan penyedia otomatisasi seperti pengecekan mutasi ke bank-bank poluler seperti: BCA, MANDIRI, BSM, BRI, BNI (dan terus berkembang), serta menyediakan layanan SMS Gateway.

Kenapa pakai SMS? Karena tidak semua orang akan memeriksa email nya, jadi SMS akan lebih bernilai bagi bisnis Anda daripada mengirimkan notifikasi via E-Mail

Perpaduan Woocomerce dengan SMS Gateway ini akan membuat bisnis Anda lebih bernilai dan mendapatkan feedback lebih dari pelanggan.

== Installation ==

Sebelum memakai plugin ini pastikan Anda telah memiliki ioncube terinstall pada module PHP Anda.

Pastikan Anda sudah menginstall Billingotomatis SMS Gateway sebelum menginstall plugin ini.
